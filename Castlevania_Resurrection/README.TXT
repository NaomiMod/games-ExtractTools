HOW TO USE

------------


1) Use GD-ROM explorer to open .GDI and extract all files in any folder of your choiche.

2) Extract this zip inside extracted game folder

3) Run CVR_TEXTURES_EXTRACTOR.bat


The script will create a "Texture" folder containing PVR stored in own folders.
